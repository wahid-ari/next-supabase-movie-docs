# Update Director

To edit the data in the specified collection with the specified parameters.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge variant="info">PUT</Badge>

**URL :** <Badge variant="dark">/api/director?id=(id)</Badge>

**Body :** <Badge variant="purple">JSON</Badge>

```json
{
	"name": "New Director Edit",
	"image_url": "https://www.themoviedb.org/t/p/w300_and_h450_bestv2/82W339V8turXUdaCajqOyekxhmD.jpg",
	"gender": 1,
	"biography": "New Director Bio Edit",
	"country_id": 1
}
```

<Param name="name" type="string" required={true}>
	Name of director.
</Param>
<Param name="image_url" type="string">
	Image URL of director.
</Param>
<Param name="gender" type="integer">
	Gender of director. 1 (Male) | 2 (Female)
</Param>
<Param name="biography" type="string">
	Biography of director.
</Param>
<Param name="country_id" type="integer">
	Country ID of director
</Param>

**Response :** <BadgeOutline variant="success">200 - OK</BadgeOutline>

```json
{
  "message": "Success update director"
}
```

**Response :** <BadgeOutline variant="info">422 - Unprocessable Content</BadgeOutline>

```json
{
  "error": "Name required"
}
```

**Response :** <BadgeOutline variant="warning">404 - Not Found</BadgeOutline>

```json
Not Found
```

**Response :** <BadgeOutline variant="danger">500 - Server Error</BadgeOutline>

```json
Internal Server Error
```
